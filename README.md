## 使用技術
- Ruby
- Ruby on Rails 
- PosgreSQL
- Heroku
- AWS (S3)

- ユーザー登録、ログイン機能(device)
- 画像投稿(CarrierWave、MiniMagick)
- いいね機能(Ajax)
- コメント機能(Ajax)
